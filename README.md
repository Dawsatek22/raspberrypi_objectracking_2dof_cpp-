# raspberrypi_objectracking_2dof_cpp-
The second version of a Raspberry Pi c++ object tracking robot(link is here: https://github.com/Dawsatek22/raspberrypi_objectracking_1dof_cpp). This version  can now use the x and y direction.Tutorials and cad list are coming soon but if you wanna use the code earlier here are links to recommended software you need: opencv c++ version for the Raspberry Pi ( https://opencv.org/get-started/?utm_source=opcv&utm_medium=home ),lccv for Raspberry Pi camera ( https://github.com/kbarni/LCCV.git ),
PIPCA9685 software for the Raspberry Pi ( https://github.com/barulicm/PiPCA9685.git ),
vnc viewer for pc/laptop ( https://www.realvnc.com/en/connect/download/viewer/?lai_sr=0-4&lai_sl=l&lai_p=1&lai_na=488119 ),
